## 一个跨平台的 Chat GPT GUI
* 项目架构：基于tauri 及 angular
* 精细的UI：支持多dark、light主题，整体风格遵循Github官网的风格指南
* 多端支持：支持 Mac 桌面应用，PC及移动应用（网页），支持使用手机浏览器访问
* 体积够小：整个应用是有10MB左右
* 自动升级：支持应用自动升级，每次启动应用会检测是否有最新版本，用户选择性升级
